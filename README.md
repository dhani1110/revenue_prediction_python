# revenue_prediction_python

## Overview
This project predicts monthly revenue for products using past sales data.

## What I did
- Loaded and merged sales data
- Calculated revenue using quantity and price
- Converted order dates into monthly data
- Used previous month revenue to predict current month revenue
- Built a simple prediction model in Python
- Compared actual vs predicted revenue

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
